Button Bar

Note that the Analysis buttons (Track/Album) and Gain buttons (Track/Album/Constant) are switchable by using the small arrow to the immediate right of the button. For example, if the Track Gain button is visible, you can choose either the Album Gain or Constant Gain buttons by clicking on the arrow and making your selection.

The Button Bar has the following icons:


Add File(s)		 This will bring up a File browse dialog that lets you navigate to a folder and then add one or more files from that folder to the File List.

Add Folder		 This will bring up a Directory browser that lets you navigate to a folder and then select it. All files in that folder will be added to the File List. If Add Subfolders is checked under Options, then all folders under the selected one will also be selected and their files added to the File List.

Track Analysis		 This will analyze each file in the File List and calculate the file's volume level.

Album Analysis		 This will analyze each file in the File List as part of an album and calculate the total album volume level.

Track Gain		 This will correct all files in the File List using the Target Volume in Track mode.

Album Gain		 This will correct all files in the File List using the Target Volume in Album mode.

Constant Gain		 This will adjust all files in the File List using the dB value indicated by the slider in the Constant Gain Change dialog.

Clear Files		 This will remove selected files from the File List.

Clear All		 This will remove all files from the File List.
